Welcome to the Cookie POC

Clicking here performs a GET on the /redirect endpoint with a query parameter url and value "/". The /redirect responds with 302 - Found with the given url query parameter as the Location (= "/"). The response contains 2 cookies: cxrsProcessed="true" which expires after 60 seconds and survivor="kilroy" which expires after 24 hours.

The response goes through Apache which sets the expiration date of the cxrsProcessed cookie to a date in the past (Thu, 01 Jan 1970 00:00:00 GMT). Upon receipt of this header in the response, browsers should remove the cxrsProcessed cookie. Other cookies must not be affected.
